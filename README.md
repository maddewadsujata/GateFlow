# GateFlow - Hostel GatePass & GaVPass Management System

A complete Android application for managing hostel gate passes and home leaves using Java, XML, and Firebase.

## Features

- Student signup with OTP verification
- Staff management (Warden, HOD, Principal)
- GatePass and GaVPass application system
- Real-time request tracking
- Firebase Cloud Messaging notifications
- Watchman verification dashboard
- QR code generation for passes
- Material Design UI with dark mode
- RecyclerView for efficient list management

## Tech Stack

- **Language**: Java
- **UI**: XML, Material Design
- **Backend**: Firebase (Auth, Realtime DB, Storage, Cloud Messaging)
- **Android SDK**: Min 21, Target 34
- **Build System**: Gradle

## Quick Start

1. Clone repository
2. Open in Android Studio
3. Add google-services.json
4. Sync Gradle
5. Run on emulator or device
